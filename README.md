# StockCut

StockCut is a local-first free tool site for sheet cutting, linear cutting, saw kerf calculation, printable layouts, CSV/JSON export, localStorage autosave, and SEO preset pages.

Run:

```bash
npm install
npm run lint
npm run build
```

MVP scope follows the two Chinese planning documents in this repository. The tool intentionally does not include accounts, cloud save, CNC, DXF, G-code, non-rectangular nesting, circular/triangular/polygonal parts, enterprise inventory, order scheduling, AI cabinet design, or angle cutting.
