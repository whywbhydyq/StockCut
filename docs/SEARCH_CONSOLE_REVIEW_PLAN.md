# StockCut Search Console review plan

Use this file after the site has Google Search Console data. Do not create new long-tail URLs from guesses alone.

## Week 4

Export Performance > Search results by Query and Page for the last 28 days.

- Impressions with no clicks: adjust title and first-screen promise first.
- Positions 10-40: strengthen examples, preset modules, and canonical internal links.
- No impressions: check URL Inspection, sitemap inclusion, canonical, and internal links.

## Week 8

Compare last 28 days with the previous 28 days.

- If the same query appears for multiple plywood pages, assign one canonical target and weaken or merge the other page.
- If a specific preset query appears repeatedly, improve that existing preset page before creating a new URL.
- If a page gets clicks but no tool use, improve the input/result path before adding more copy.

## Week 12

Decide whether to expand, merge, or hold.

- Expand only when a query has repeated impressions and a distinct task.
- Merge when two pages compete for the same query and neither has a unique task.
- Hold when impressions are still too low; do not keep changing TDK without data.
