# StockCut Keyword Cluster Map — Offline `seo-cluster` Execution

This map consolidates source-declared intent clusters. It does not use live keyword volume, CPC, SERP scraping, or competitor exports.

## Cluster 1 — Sheet goods and panel layouts

Primary intent: users need a layout for rectangular parts on sheet stock.

Canonical pages:

- `/sheet-cutting-optimizer`
- `/4x8-plywood-cut-list-optimizer`
- `/plywood-cutting-layout-calculator`
- `/mdf-sheet-cut-calculator`
- `/melamine-cut-list-optimizer`
- `/acrylic-sheet-cutting-layout-tool`
- `/cabinet-cut-list-optimizer`

Expansion rule: add a new page only when the material or workflow has distinct constraints that cannot be covered by an existing page section.

## Cluster 2 — Linear stock cutting

Primary intent: users need an ordered cut list for boards, pipe, extrusion, tube, rebar, or lumber.

Canonical pages:

- `/linear-cutting-optimizer`
- `/linear-bar-cutting-list-optimizer`
- `/pvc-pipe-cutting-optimizer`
- `/steel-tube-cutting-optimizer`
- `/aluminum-extrusion-cut-list-optimizer`
- `/rebar-cutting-optimizer`
- `/lumber-length-cutting-optimizer`

Expansion rule: do not split synonyms into separate pages unless GSC evidence shows materially different query behavior.

## Cluster 3 — Kerf, fit, waste, and edge treatment

Primary intent: users are diagnosing why nominal dimensions do not fit or why waste appears higher than expected.

Canonical pages:

- `/how-to-account-for-saw-kerf`
- `/saw-kerf-calculator`
- `/saw-kerf-compensation-calculator`
- `/saw-kerf-explained`
- `/why-two-24-inch-panels-do-not-fit-on-a-48-inch-sheet`
- `/edge-banding-in-cut-list`
- `/reduce-plywood-waste`

Expansion rule: educational pages should link back to tool pages and should not duplicate calculator UI promises.

## Cluster 4 — Methodology, comparisons, and workflow decisions

Primary intent: users compare approaches or want to understand algorithmic boundaries.

Canonical pages:

- `/cut-list-optimization-methodology`
- `/guillotine-cut-vs-nesting`
- `/cut-list-optimizer-vs-excel`
- `/cut-list-optimizer-vs-sketchup`
- `/how-to-read-a-plywood-cutting-diagram`
- `/grain-direction-in-cut-lists`
- `/plywood-factory-edge-trim`

Expansion rule: comparison pages should stay factual and boundary-driven. Do not claim superiority without testable criteria.

## Change-control rule

New cluster pages require a proposed SEO change manifest unless they are purely governance or documentation additions.
